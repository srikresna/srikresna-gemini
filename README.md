This is a srikresna gemini LLM Chatbot. This app is powered by Google's Gemini Generative AI models. This app is built using Streamlit and hosted on Streamlit Share.

## Feature
- Multi turn chatbot
- Image captioning
- Chat with PDF
- Temperature output control
